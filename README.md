# Тестовый проект по автоматизации QA

Автотесты для учебного интернет-магазина [SauceDemo](https://www.saucedemo.com/), написанные на Python с использованием Selenium WebDriver.

---

## Что проверяют тесты?

1. **Успешный вход** — стандартный пользователь попадает в каталог товаров.
2. **Ошибка при неверном пароле** — система показывает предупреждение.
3. **Добавление товара в корзину** — в корзине появляется значок с количеством товаров.

---

## Технологии

- **Python** 3.x  
- **Selenium WebDriver**  
- **ChromeDriver** (для браузера Google Chrome)  
- **Git** (для контроля версий)

---

## Как запустить тесты на своем компьютере?

1. **Склонируй репозиторий:**
   ```bash
   git clone https://github.com/RoninDante/qa-auto-tests.git
   ```
   
Перейди в папку с проектом:

```bash
cd qa-auto-tests
```
Установи зависимости (потребуется только один раз):

```bash
pip install -r requirements.txt
```
Запусти тест:

```bash
python test_login.py
```
Никаких дополнительных настроек не требуется.
